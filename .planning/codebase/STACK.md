# Technology Stack

## Core
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **UI Library:** React 18

## Styling
- **CSS Framework:** Tailwind CSS
- **Animations:** Framer Motion
- **Base Styles:** Global CSS (`src/app/globals.css`)

## Utilities & Libraries
- **Google APIs:** `googleapis` (Drive v3, Docs v1)
- **Authentication:** `google-auth-library` (OAuth2)
- **File Processing:** `sharp` (Image processing)
- **ID Generation:** `uuid`

## Infrastructure
- **Platform:** Vercel (assumed based on `vercel.json`)
- **Storage:** Google Drive (used as primary database and file storage)
